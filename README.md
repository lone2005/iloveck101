iloveck101
----------

前端也要"硬"起來 !!!!!

向大大們致敬 !

（目前只測過 Chrome, Firefox 應該也沒問題）

## 如何安裝

    git clone git@github.com:poying/iloveck101.git
    cd iloveck101
    npm install
    bower install
    grunt

## 如何使用

執行

    npm start

然後打開 Browser 輸入 `http://localhost:3000`
